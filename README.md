# ⚡ Controle e Programação de Obra no Setor Elétrico

Sistema de gestão operacional desenvolvido em **Excel + VBA** para o acompanhamento e programação de obras de construção e manutenção em redes elétricas.

---

## 📋 Sobre o Projeto

Este arquivo centraliza toda a gestão operacional de uma base para construção e manutenção de redes elétricas. Ele integra o planejamento e a execução de obras contratadas por uma distribuidora de energia, possibilitando o acompanhamento em tempo real de metas financeiras por equipe, programações diárias, status de execuação, eficiência operacional e calendário de atividades — tudo sem necessidade de sistemas externos.

---

## 🏗️ Estrutura das Abas

| Aba | Função |
|---|---|
| `DashBoard` | Painel gerencial com indicadores financeiros, metas e eficiência por equipe |
| `Programações` | Registro histórico e futuro de todas as atividades programadas |
| `Carteira` | Portfólio completo de obras com rastreamento financeiro e de status |
| `Calendário` | Grade mensal de alocação de equipes por obra com status visual |
| `Auxiliar` | Tabelas de referência, cadastros e catálogo de serviços |
| `PS` | Formulário de Plano de Segurança gerado por VBA |
